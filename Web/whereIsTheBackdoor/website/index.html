API description

  • /test: just a GET test endpoint.
      PARAMS
    • param: optional. Is printed.

  • /db_connection: performs a basic request to the database.
      NO PARAMS

  • /server_health: checks the server status. Runs a file integrity test, and checks if hosts are up.
      PARAMS
    • timeout: optional. The timeout in ms set for each test. Default is no timeout.

    • Usage example: /test?param=hello
